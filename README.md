This exercise was the second weekly assignment for the course **Mobile programming with Native Technologies**.

## Features
- Centered and emphasized "Login" title
- Username and password input fields
- Icons in the username (email) and password fields
- Material-style "Submit" button

## Building and Running
You can run the project in Android Studio:

1. Open Android Studio
2. Choose **Open** and select this folder (`LoginForm`)
3. Let Gradle synchronize
4. Choose a device/emulator and press **Run** (the green play button)

## Code
The main UI is located in:
- `app/src/main/java/com/example/loginform/MainActivity.kt`

The login form is implemented in the `LoginForm` composable function using Material 3 components (`Text`, `OutlinedTextField`, `Button`, `Icon`).
